Equilibar EPR-3000 electronic pressure regulator¶

Equlibar EPR-3000 electronic pressure regulator

Note this plugin requires the following information to be present in .place.cfg:

serial_port = enter_value_here                            #(e.g. /dev/ttyUSB0)
maximum_pressure = enter_value_here                       #(e.g. 20.0)
minimum_pressure = enter_value_here                       #(e.g. 0.0)
tolerance = enter_value_here                              #(e.g. 0.02)
pressure_increase_minimum = enter_value_here              #(e.g. 0.01)
pressure_increase_minimum_time_period = enter_value_here  #(e.g. 5)
pressure_decrease_minimum = enter_value_here              #(e.g. 0.02)
pressure_decrease_minimum_time_period = enter_value_here  #(e.g. 100)
max_adjustment_wait_time = enter_value_here               #(e.g. 120)

The Equilibar EPR 3000 class

The EquilibarEPR3000 module requires the following configuration data (accessible as self._config[’key’]):

Key

Type

Meaning

units

string

the pressure units to use. Either “MPa” or “psi”

start_pressure

float

the pressure at the first update. This is also currently the pressure for all updates.

end_true

bool

True to set the pressure to a new value at the end of the experimnet

end_pressure

float

the pressure to set the regulator to at the end of the experiment if end_true == True

The EquilibarEPR3000 module will produce the following experimental metadata:

Key

Type

Meaning

EquilibarEPR3000-units

string

the pressure units used by the regulator

The EquilibarEPR3000 will produce the following experimental data:

Heading

Type

Meaning

pressure

(current_pressure,setpoint) array of float64

the pressure and setpoint from the regulator

Note

PLACE will usually add the instrument class name to the heading. For example, pressure will be recorded as EquilibarEPR3000-pressure when using the regulator. The reason for this is because NumPy will not check for duplicate heading names automatically, so prepending the class name greatly reduces the likelihood of duplication.
